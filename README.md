# HTML & CSS Layout Practice

This repository contains two simple web layout projects built using **only HTML and CSS**.  
Each project focuses on mastering a different CSS layout technique — **Flexbox** and **Grid**.

---

##  Projects Overview

###  App 1 — Flexbox Layout
- Built mainly using **CSS Flexbox** for structure and alignment.
- Focuses on understanding **flex-direction**, **justify-content**, and **align-items**
- Great for practicing responsive design using Flexbox.

### App 2 — Grid Layout
- Built mainly using **CSS Grid** to create 2D layouts.
- Focuses on concepts like **grid-template-columns**, **grid-template-rows**, **grid-area**, and **span**.
- Ideal for learning how to design complex, magazine-style layouts.

---

##  Tech Stack
- **HTML5**
- **CSS3 (Flexbox & Grid)**

---

##  Purpose
These projects were created **purely for personal learning and practice** —  
to understand the differences and use-cases of **Flexbox vs Grid** in modern web design.
